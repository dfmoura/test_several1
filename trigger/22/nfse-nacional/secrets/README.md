# Certificado A1 (local / Docker)

1. Copy your `.pfx` here as `certificado.pfx`
2. Optional: create `certificado.senha` with the PFX password on one line (or set `NFSE_CERT_PASSWORD` in `.env` instead)
3. Never commit files in this folder after placing `certificado.pfx here. Close Docker first if containers use the cert.
